## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/brodluck/Data-Analyzer-for-WhatsApp-Chats.git
   ```

2. **Install Dependencies:**
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Run the Application:**
   ```bash
   streamlit run server.py 
   ```
